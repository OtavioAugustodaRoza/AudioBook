##  Funcionalidades

- 🎵 Tocar, pausar e trocar faixas  
- 🎚️ Controle de volume  
- ⏩ Avançar e retroceder música  
- 🕓 Barra de progresso dinâmica  
- 🖼️ Capa, título e artista da música  
- 📱 Layout totalmente responsivo com Tailwind
